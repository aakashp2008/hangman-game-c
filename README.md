# Hangman Game

## 📝 About the Project
The Hangman Game is a beginner-friendly C application that allows users to play the classic word guessing game in a console environment. The player guesses one letter at a time to reveal the hidden word before running out of attempts.

## ✨ Features
- Guess letters one at a time
- Display hidden word progress
- Limited number of incorrect attempts
- Win and lose detection
- Simple console-based interface

## 🛠️ Technologies Used
- C Programming
- Strings
- Character Arrays
- Functions
- Loops
- Conditional Statements

## 🎯 Learning Outcomes
This project helped me learn:
- Working with strings and character arrays
- Implementing game logic
- Using loops and conditional statements
- Handling user input
- Developing interactive console applications

## 🚀 Future Improvements
- Add multiple difficulty levels
- Load words from a file
- Display ASCII Hangman graphics
- Add score tracking
- Support multiplayer mode

## 📖 About
A simple C-based Hangman Game where players guess letters to reveal a hidden word using strings, loops, and console-based gameplay.
